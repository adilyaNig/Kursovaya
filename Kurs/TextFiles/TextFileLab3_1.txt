1.3. Лабораторная работа №3. Изменение двумерного массива
Задача №1. Дан двумерный массив. Все положительные элементы массива  уменьшить на 20, а отрицательные – заменить на 5.
Решение:
С помощью двух вложенных циклов обрабатывается каждый элемент массива. Если элемент x[i,j] больше 0, то из него вычитается 20. Если элемент меньше 0, то его значение устанавливается равным 5. Если элемент равен 0, то ничего не происходит (это условие не обрабатывается). Таким образом, все положительные значения уменьшаются на 20, а все отрицательные значения заменяются на 5. Программа выводит обработанный массив на экран. Внешний цикл проходит по строкам, а внутренний — по столбцам. Для удобства чтения используется табуляция.
Фрагмент программного кода: 
double[,] x = new double[n, m];
for (i = 0; i < n; i++)
{	for (j = 0; j < m; j++)
{
if (x[i, j] > 0)
x[i, j] -= 20;
else
{	if (x[i, j] < 0)
x[i, j] = 5;
}
}
}
Console.WriteLine("результат:");
for(i = 0; i < n; i++)
{
for(j = 0; j < m;j++)
{
Console.Write(x[i, j] + "\t");
}
Console.WriteLine();
}
