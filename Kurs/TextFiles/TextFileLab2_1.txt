1.2.	Лабораторная работа №2. Поиск в двумерном массиве
Задача №1. Двумерный массив размера n*10 состоит из случайных целых чисел в диапазоне от -50 до 100. В столбце с номером k найти минимальный элемент.
Решение: 
k — номер столбца, в котором будет производиться поиск минимального элемента.
min — переменная для хранения минимального значения в указанном столбце.
С помощью вложенного цикла for каждый элемент массива x[i,j] заполняется случайным целым числом в диапазоне от -50 до 100. Для генерации случайных чисел используется объект rnd класса Random и  его метод Next(-50,101), который выдает случайные целые числа в диапазоне      [-50,101). Для поиска минимального значения в столбце с номером  k  инициализируется переменная min, которой присваивается значение первого элемента в этом столбце (то есть элемент с индексами [0, k]). Это значение служит начальным ориентиром для поиска. 
   Открывается цикл for, который начинается с индекса 1, так как первое значение уже сохранено в переменной min. Внутри этого цикла используется условный оператор if, который проверяет, меньше ли текущий элемент в столбце  k  (то есть x[i, k]) минимального значения (min). Если это условие истинно, то переменной min присваивается текущее значение элемента. После завершения поиска минимального значения в указанном столбце на экран выводится результат.
Программный код:
int n,  i;
Console.Write("n=");
n=Convert.ToInt32(Console.ReadLine());
int[,] x = new int[n,10];
Random rnd=new Random();
for(i=0;i<n;i++)
{ for(j=0;j<10;j++)
    { x[i,j]=rnd.Next(-50,101); }
}
for (i = 0; i < n; i++)
{
for (j = 0; j < 10; j++)
{ Console.Write(x[i, j] + "\t"); }
Console.WriteLine();
}
int min;
Console.Write("k=");
int k=Convert.ToInt32(Console.ReadLine());
min = x[0, k];
for(i = 1; i < n;i++)
{
if (x[i, k] < min)
{ min = x[i, k]; }
}
Console.WriteLine($"минимальный {min}");
