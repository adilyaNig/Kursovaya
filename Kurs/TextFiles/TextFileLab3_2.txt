Задача №2. Дан двумерный массив n*m. Поменять местами максимальный и минимальный элементы в массиве.
Решение: 
int k, l - индексы максимального элемента в двумерном массиве,
int min_i, min_j - индексы минимального элемента в двумерном массиве,
double max, min - значения максимального  и минимального элементов
Переменным max и min присваиваются значения первого элемента массива x (max = x[0, 0]; min = x[0, 0];). Аналогично, инициализируются значением 0 индексы k, l, min_i и min_j. В цикле for проверяется, имеет ли текущий элемент значение больше, чем значение переменной max. Если условие выполняется, переменной max присваивается значение элемента, а в переменные k и l сохраняются индексы текущего элемента. Если текущий элемент имеет значение меньше, чем значение переменной min, то обновляется значение min, а также сохраняются индексы текущего элемента в переменные min_i и min_j.
Далее максимальный элемент заменяется (по индексам k и l) на значение переменной min, а минимальный элемент (по индексам min_i и min_j) на значение переменной max. Измененный массив выводится на экран.
Фрагмент программного кода:
double[,] x = new double[n, m]; 
max = x[0, 0];
min = x[0, 0];
k = 0; l = 0;
min_i = 0; min_j = 0;
for (i = 0; i < n; i++)
{	for( j = 0; j < m; j++)
{	if (x[i, j] > max)
{	max = x[i, j];
k = i;
l = j;	
}
if (x[i, j] < min)
{	min = x[i, j];
min_i = i;
min_j = j;	
}
}
}
x[k,l] = min;
x[min_i, min_j] = max;
Console.WriteLine("результат:");
for (i = 0; i < n; i++)
{	for(j = 0; j < m; j++)
{
Console.Write(x[i,j]+"\t");
}
Console.WriteLine();
}
