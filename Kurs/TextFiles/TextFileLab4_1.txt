1.4. Лабораторная работа №4.  Формирование двумерного массива
Задача №1. Сформировать массив размера m*m, где на главной диагонали расположены единицы, выше – тройки, а ниже главной диагонали – двойки.
Решение: 
Создается двумерный массив x размером m*m, который будет хранить целые числа. Если индекс строки равен индексу столбца (i == j), то мы элемент находится на главной диагонали и ему присваивается значение единица. Если индекс строки меньше индекса столбца (i< j), то элемент находится выше главной диагонали и получает значение три, иначе элемент находится ниже главной диагонали (i > j), в этом случае элемент x[i, j] устанавливается равным 2.
Программный код:
int[,] x = new int[m, m];
for(i = 0; i < m; ++i)
{	for(j = 0; j < m; ++j)
{	if(i==j)
{ x[i, j] = 1; }
else if(i<j)
{ x[i, j] = 3; }
else
{ x[i, j] = 2;}
}
}
Console.WriteLine("вывод массива:");
for (i = 0; i < m; i++)
{	for(j = 0; j < m; j++)
{ Console.Write(x[i,j]+"\t"); }
Console.WriteLine();
}
