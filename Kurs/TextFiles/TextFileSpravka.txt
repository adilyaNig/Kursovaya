-Двумерные массивы
	Представляют собой таблицы. Каждый элемент имеет два индекса, первый определяет номер строки, второй – номер столбца, на пересечении которых находится элемент. Нумерация строк и столбцов начинается с нуля. 
	Двумерный массив размера n*m имеет вид:
	x[0,0]     x[0,1]     x[0,2]   …  x[0,m-1]
	x[1,0]     x[1,1]     x[1,2]   …  x[1,m-1]
	…

	x[n-1,0]     x[n-1,1]     x[n-1,2]     …     x[n-1,m-1]
	x – имя массива,
	n – количество строк,
	m – количество столбцов

Объявление и создание двумерного массива
	тип [ , ] имя_массива = new тип [размер1, размер2]; //размер 1 – количество строк, размер 2 – количество столбцов
	Например:
	int [ , ] a = new int [3, 4];

Объявление двумерного массива с инициализацией:
	тип [ , ] имя_массива = {{элементы 1-ой строки}, … , {элементы n- ой строки}, … , {элементы n-ой строки}};
	Например:
	int [ , ] a = {{0,1,2}, {3,4,5}};

Ввод и вывод двумерных массивов
(вложенный цикл)
	Ввод двумерного целочисленного массива размера n*m, используя циклы с предусловием:
	n=2,
	m=3;
	15     -10     0
	 4       38    100
	
	Двумерный массив можно вводить построчно или по столбцам. Если массив вводится построчно, то внешний цикл организуется по номеру строки, а внутренний – по номеру столбца.
	Если массив вводится по столбцам, то внешний цикл организуется по номеру столбца, а внутренний по номеру строки.

	// i – номер строки, n – строка, j – номер столбца, m – столбец.
	int n, m, i, j;
	Console.Write(“n=”);
	n=Convert.ToInt32(Console.ReadLine());
	Console.Write(“m=”);
	m=Convert.ToInt32(Console.ReadLine());
	int [ , ] a = new int [n, m];
	i=0;
	while (i< n)
	{
		j=0;
	while (j<m) //когда j=m, то выход из цикла
	{
		a [i, j] = Convert.ToInt32(Console.ReadLine());
		j++;
	}
	i++;
	}	//Здесь организован построчный ввод массива a

Ввод двумерного массива циклами с параметром:
for(i=0; i<n; ++i)
{  
	for(j=0; j<m; ++j)
	{
		Console.Write(“a[{0},{1}]=”,i,j);
		a[i,j] = Convert.ToInt32(Console.ReadLine());
	}
}

Вывод двумерного массива в виде матрицы:
for (i=0; i<n ; i++)
{
	for (j=0; j<m; j++)
	{
		Console.Write($”{a[i,j]}\t”);
	}
	Console.WriteLine();
}
