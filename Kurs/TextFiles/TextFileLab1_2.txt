Задача №2. Дан двумерный массив размера n*n. Вычислить среднее арифметическое отрицательных элементов, лежащих на главной диагонали.
Решение:
n – количество строк и столбцов в двумерном массиве, 
k - количество отрицательных элементов  главной диагонали массива,
s - сумма отрицательных элементов  главной диагонали.
Первоначально  сумма и количество отрицательных элементов  главной диагонали массива равны нулю. Элемент двумерного массива расположен на главной диагонали, если его номер строки совпадает с номером столбца, т.е. a[i,i] – элемент главной диагонали. Таким образом, для обработки диагональных элементов двумерного массива достаточно организовать цикл по одной переменной i. Если элемент главной диагонали (a[i,i]) отрицательный, то количество таких элементов (k++) увеличивается на единицу, а элемент добавляется в  сумму (s += a[i,i]).
После завершения цикла проверяется, были ли найдены отрицательные элементы главной диагонали (k не равен 0). Если такие элементы есть, то на консоль выводится среднее арифметическое этих элементов (s/k). Если же отрицательных элементов не найдено, то на консоль выводится сообщение о том, что таких элементов нет.
Программный код:
int n,  i;
Console.Write("n=");
n=Convert.ToInt32(Console.ReadLine());
double[,] a = new double[n,n];
for(i=0;i<n;i++)
{
	for(j=0;j<n;j++)
	{
		Console.Write($"a[{i},{j}]=");
		a[i,j]=Convert.ToDouble(Console.ReadLine());
	}
}
int k = 0; double s=0;
for(i=0;i<n;i++)
	{
		if (a[i,i]<0)
		{ k++; s += a[i,i]; }
	}
if (k != 0)
{ Console.WriteLine($"Среднее арифметическое: {s / k}"); }
else
{ Console.WriteLine("Нет отрицательных элементов"); }
