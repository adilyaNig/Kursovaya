1.1.	Лабораторная работа №1. Вычисление сумм, произведений и количества элементов в двумерном массиве

Задача №1. Дан целочисленный двумерный массив размера n*m. В третьей строке массива вычислить количество и произведение четных элементов.
Решение: 
Объявляются переменные целочисленного типа: 
n – количество строк в двумерном массиве, 
m – количество столбцов, 
i – порядковый номер строки, 
j – порядковый номер столбца. 
Переменная k будет использоваться для подсчета количества четных элементов в третьей строке массива. Первоначально количество таких элементов равно нулю. Переменная p будет использоваться для вычисления произведения четных элементов в третьей строке массива. Первоначально произведение таких элементов равно единице. В цикле for по параметру j перебираются все столбцы, начиная со столбца с номером 0 до столбца с номером m. Условие a[i,j] % 2 == 0 проверяет, является ли элемент массива a[i,j] четным (остаток от деления a[i,j] на 2 равен 0). Если условие соблюдается, то найден четный элемент массива и количество четных элементов увеличивается на единицу (k++), а текущее произведение (p) умножается на этот элемент a[i,j]. Когда поиск по всем столбцам закончится, в консоль выводится количество и произведение четных элементов третьей строки массива.
Программный код:
int n, m, i, j;
Console.Write("n=");
n=Convert.ToInt32(Console.ReadLine());
Console.Write("m=");
m=Convert.ToInt32(Console.ReadLine());
int[,] a = new int[n,m];
for(i=0;i<n;i++)
{	for(j=0;j<m;j++)
	{	Console.Write($"a[{i},{j}]=");
		a[i,j]=Convert.ToInt32(Console.ReadLine());
	}
}
int k = 0;
double p=1;
i = 2;
for(j=0; j<m; j++)
{
	 if (a[i,j] % 2 == 0)
	{ k++;  p *= a[i,j]; }
}
Console.WriteLine($"Количество - {k} Произведение - {p}");
