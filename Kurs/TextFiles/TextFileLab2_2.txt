Задача №2. Дан двумерный массив размера n x m.  Встречается ли в этом  массиве элемент, больший 5?
Решение:
Создается двумерный массив x размера  n*m, каждый элемент которого пользователь самостоятельно заполняет числами. Задается логическая переменная found для хранения информации о том, найден ли элемент со значением больше 5. Первоначально found=false, т.е. нужный элемент не найден. С помощью цикла for перебираются все элементы массива: для цикла по переменной i условие продолжения цикла имеют вид:  i < n  && !found, для цикла по j – условие будет  j< m  && !found. Если при просмотре двумерного массива найден элемент, больший 5, то f:=true и осуществляется выход из цикла, т.к.  !found будет false и все составные условия i < n  && !found и j< m  && !found будут false .
Если переменная found имеет значение true, то элемент, больший пяти, в массиве найден  и выводится сообщение об этом, в противном случае – сообщение об отсутствии таких элементов.
Программный код:
int n, m, i, j;
            Console.Write("n=");
            n = Convert.ToInt32(Console.ReadLine());
            Console.Write("m=");
            m = Convert.ToInt32(Console.ReadLine());
            int[,] x = new int[n, m];
            for (i = 0; i < n; i++)
            {
                for (j = 0; j < m; j++)
                {
                    Console.Write($"x[{i},{j}]=");
                    x[i, j] = Convert.ToInt32(Console.ReadLine());
                }
            }
            for (i = 0; i < n; i++)
            {
                for (j = 0; j < m; j++)
                { Console.Write(x[i, j] + "\t"); }
                Console.WriteLine();
  }
bool found = false;
            for (i = 0; i < n  && !found; i++)
            {
                for (j = 0; j < m  && !found; j++)
                {
                    if (x[i, j] > 5)
                        found = true;
                }
            }
if (found)
{  Console.WriteLine("В массиве есть элемент, больший 5."); }
else
{  Console.WriteLine("В массиве нет элементов, больших 5.");}
